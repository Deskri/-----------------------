Интенсивность

В нашем примере интенсивность входного потока заявок λo = 1. Cчитается, что все СМО имеют одинаковую среднюю длительность обслуживания заявок, т.е. V1 = V2 = V3 = V4 = V5 = 3,0.
Теперь необходимо составить систему уравнений.

λ0 = P10 * λ1

λ1 = P31 * λ3

λ2 = P42 * λ4

λ3 = P13 * λ1 + P23 * λ2 + P53 * λ5

λ4 = P34 * λ3

λ5 = P25 * λ2 + P45 * λ4 + P55 * λ5
